CameraEx ALL 7 APPS

com.sony.scalar.hardware.CameraEx

The primary camera control class wrapping Sony's proprietary camera hardware. Extends android.hardware.Camera with burst capture, direct shutter, gamma table LUTs, color select for partial color effects, and hardware-accelerated focus/aperture/shutter control.

Opening the Camera

// Standard camera open pattern (used by ALL 7 reference apps)
CameraEx.OpenOptions opts = new CameraEx.OpenOptions();
opts.setPreview(true);              // Enable live preview on LCD/EVF
opts.setInheritSetting(true);       // Inherit user's current camera settings
opts.setRecordingMode(0);           // 0 = still photo, 1 = movie
opts.setTargetMedia(AvindexStore.getExternalMediaIds()[0]);
CameraEx cameraEx = CameraEx.open(0, opts);

// Get underlying Android Camera for compatibility
Camera normalCamera = cameraEx.getNormalCamera();

// Reopen with different settings (e.g., switch to movie mode)
CameraEx.OpenOptions newOpts = new CameraEx.OpenOptions();
newOpts.setRecordingMode(1);
cameraEx.reopen(newOpts, new CameraEx.OpenCallback() {
    public void onReopened(CameraEx camera) {
        // Camera is now in movie mode
    }
});

Color Select API picture-effect-plus

Hardware partial color extraction — keeps selected colors while desaturating everything else.

// Sample color from live preview at screen coordinates

// Assign to channel 0 (supports 2 independent channels)

// Enable extraction: bitmask 1=ch0, 2=ch1, 3=both

Gamma Table API live-view-grading, smooth-reflection

Custom gamma tables for complete tone curve control. Used for color grading presets and ND filter simulation.

// Create and apply custom gamma table from CSV
CameraEx.GammaTable table = cameraEx.createGammaTable();
table.setPictureEffectGammaForceOff(true);  // Override built-in gamma

InputStream csv = getAssets().open("std_vivid.csv");
table.write(csv);  // 1024 x 16-bit tone curve values
csv.close();

cameraEx.setExtendedGammaTable(table);  // Apply to live view + capture
table.release();

// Restore default: pass null
cameraEx.setExtendedGammaTable(null);

DSP 5 APPS

com.sony.scalar.hardware.DSP

Hardware Digital Signal Processor for image compositing, blending, color conversion, and effects. Operates on raw data or OptimizedImage buffers in dedicated image processing hardware.

// DSP usage pattern
DSP dsp = DSP.createProcessor("sony-di-dsp");
dsp.setProgram("libsa_smoothphotography.so");

DeviceBuffer boot = dsp.createBuffer(60);
DeviceBuffer params = dsp.createBuffer(256);

int desc = dsp.getPropertyAsInt("program-descriptor");
int addr = dsp.getPropertyAsInt(image, "memory-address");

dsp.setArg(0, boot);
dsp.setArg(1, params);
boolean ok = dsp.execute();

dsp.clearProgram();
dsp.release();

OptimizedImage

com.sony.scalar.graphics.OptimizedImage

Sony's proprietary image container in hardware-accessible memory (YUV422, 2 bytes/pixel). Never constructed directly. MUST always call release() to free hardware memory.

if (image != null && image.isValid()) {
    int w = image.getWidth();
    int h = image.getHeight();
    // ... use image ...
    image.release();  // CRITICAL: always release
}

Canvas dimensions may differ from image (stride padding). Use DSP.getPropertyAsInt(image, "image-data-offset") for offset.

Smooth Reflection

com.sony.imaging.app.smoothreflection

Simulates long-exposure photography by capturing 2-256 raw frames and averaging them via DSP hardware. Produces smooth water, light trails, and motion blur without actual long exposure.

DSP Programs

libsa_smoothphotography.so (AVIP: _avip.so) — Commands: 6=initialize accumulator, 4=accumulate frame.

Processing Pipeline

  1. Force settings: disable OIS (setAntiHandBlurMode("off")), disable DRO
  2. Lock AWB: setAutoWhiteBalanceLock(true)
  3. Remote control mode: setRemoteControlMode(true) — suppresses shutter sound between frames
  4. Optional: adjustAperture(steps) + gamma table + RGB matrix for ND simulation
  5. Per-frame: burstableTakePicture() → raw callback → DSP accumulate
  6. DefaultDevelopFilter converts averaged raw to OptimizedImage
  7. sequence.storeImage(), cleanup

Unique APIs
